ورود به حساب

برای ورود به حساب نیاز به انجام کار خاصی نیست. تنها کافیت یک نمونه از کلاس Main ایجاد کنید و شماره تلفن اکانت را به ان تحویل دهید. 


use RubikaLib\Main;
$app = new Main(989123456789); // can be empty
این کلاس 3 ورودی دارد:
property نوع داده توضیحات
phone_number int شماره تلفن. در صروتی که خالی باشد در کامندلاین یا صفحه وب درخواست میشود
app_name string این پارامتر تنها به هنگام لاگین استفاده میشود و در لیست سشن ها ذخیره میشود. نحوه جایگذاری:
"Rubika-lib($app_name) $VERSION"
settings MainSettings تنظیمات تنظیمات شخصی سازی برنامه

نکته!

در حال حاضر وارد کردن شماره تلفن فقط برای شماره های ایرانی و تنها به صورت زیر ممکن است:
* 989123456789
* 9123456789
همچنین شماره تلفن باید به صورت int وارد شود.
میتوانید نام ربات را مشخص کنید تا در صورتی که از تعداد زیاد نشست ها استفاده میکنید به راختی انها را از یکدیگر تشخیض دهید:

$app = new Main(app_name: 'my-app-1');

نکته!

نامی که انتخاب میشود تنها در مرحله ورود به حساب استفاده خواهد شد و پس از آن وارد کردن ان تاثیری ندارد.
همچنین میتوانید تنظیمات شخصی سازی را نیز اعمال کنید: 

use RubikaLib\interfaces\MainSettings;

$settings = new MainSettings;

$settings->setUserAgent('Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:129.0) Gecko/20100101 Firefox/129.0'); 
$settings->setAppType(AppType::Shad);

$app = new Main(
  phone_number: 989123456789,
    app_name: 'my-app-1'
    settings: $settings,
);
