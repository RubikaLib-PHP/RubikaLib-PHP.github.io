موارد استفاده

کتابخانه برای جداسازی ارور های خود از باقی ارور های PHP و دیگر کتابخانه های مورد استفاده از یک کلاس Exception خصوصی با نام Failure استفاده میکند.
کافیست با یک try catch ساده انهارا دریافت کنید: 


try {
    $app = new Main(9123456789);

    // ...

    $app->RunAndLoop();
    } catch (Failure $error) {
    echo $error->getMessage() . PHP_EOL;

    if ($error->obj != array()) {
        var_dump($error->obj);
        echo PHP_EOL;
    }
}
